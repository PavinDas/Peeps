
# Peeps

Responsive Social Media Application using Flutter for Mobile and Web


![Logo](https://github.com/PavinDas/Peeps/blob/master/assets/images/WhitePeeps.png)


## Tech Stack

**Client:** Flutter

**Server:** Firebase


## Features

- Email authentication
- Image sharing (From Camera & Gallery)
- Notification badge
- Search users
- Edit profile
- Responsive
